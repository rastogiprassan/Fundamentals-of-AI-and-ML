# 📊 Student Attendance Tracker

# 📌 Description
This is a Python-based project that helps students track their attendance and calculate their attendance percentage. It also warns users if their attendance falls below the required level.



 # 🚀 Features
- Mark attendance (Present/Absent)
- Calculate attendance percentage
- Display warning if attendance is below 75%
- Simple and easy-to-use interface




# 🛠️ Technologies Used
- Python



# ▶️ How to Run

1. Install Python
2. Download or clone this repository
3. Open terminal/command prompt
4. Run the file: python attendance.py
